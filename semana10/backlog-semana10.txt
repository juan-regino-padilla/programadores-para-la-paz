Backlog del proyecto — Semana 10
Historias de usuario ordenadas de mayor a menor prioridad:

1. (Alta prioridad)
Como ciudadano quiero registrar una incidencia para dejar evidencia del caso
y que quede un registro permanente del problema reportado.

2. (Alta prioridad)
Como coordinador quiero recibir una alerta en Telegram cuando llega un nuevo
reporte para poder revisarlo y darle seguimiento a tiempo.

3. (Media prioridad)
Como miembro del equipo quiero consultar el listado de reportes para hacer
seguimiento al estado de cada caso y saber cuales siguen pendientes.

4. (Media prioridad)
Como coordinador quiero poder cambiar el estado de un reporte para que el
equipo sepa si esta pendiente, en revision o resuelto.

5. (Baja prioridad)
Como ciudadano quiero recibir una confirmacion cuando mi reporte fue recibido
para saber que la informacion llego correctamente al sistema.
