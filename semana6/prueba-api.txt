Prueba de la ruta POST /registro

Se envio una peticion POST a http://localhost:3000/registro con el siguiente JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor respondio con:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que el servidor recibe correctamente los datos enviados en el cuerpo
de la peticion POST, los procesa usando req.body y los devuelve en formato JSON
junto con un mensaje de confirmacion.
